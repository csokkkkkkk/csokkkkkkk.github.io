Project: The Modern Social Problem

An HTML narrative

Style Guide

A scrolling HTML narrative site that is made of simple graphics displaying statistics and resources.

The interface of the website is inspired by apple.com, where with each scroll, smooth transitions play and has subtle moving components.

The content and graphics of the website is inspired by thanks-in-advance.com, where the website displays informative statistics and simple, but engaging animation, as we scroll.

Effect such as the locomotive scroll, round cursors, and GSAP will be implemented with the help of simple javascript to match the aesthetic.

Chhayhee Sok, 2024. DART 349.
